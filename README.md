# Go言語N本ノック

Go言語初心者による勉強用リポジトリ

## 実行方法

```bash
git clone git@github.com:SatoshiTomita/golang-100knocks.git
cd golang-100knocks
```

.devcontainerがあるので、VSCodeなどで開くとコンテナを立ち上げることができます
